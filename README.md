# Online Food Delivery Database – Oracle SQL Project

This repository contains an **Oracle SQL\*Plus 10g project** simulating a simple **online food delivery system**.

## Features
- **Tables:** `CUSTOMERS1`, `MENU1`, `ORDERS1`, `ORDER_ITEMS1` with primary and foreign keys.  
- **Sample Data:** 10 rows of realistic data for customers, menu items, and orders.  
- **Queries:** Join, aggregation, and filtering queries for retrieving customer orders, total revenue, and low-stock items.  
- **Optional Features:** Views for customer order summaries and triggers to automate stock updates.

## Tech Stack
- Oracle SQL\*Plus 10g  
- Windows 10  

## Purpose
Designed for **learning, demonstration, and practice** of Oracle SQL concepts such as table relationships, DML, DDL, joins, aggregation, and constraints.
